The plugin will help in the creation of any online store without changing your template.

== Description ==

Free plug-in will help you without a "headache" to set up shop with a basket on ajax, which will have all the necessary sales tools.

Features

Integrates to any template, using functions, shortcodes
It does not change the structure and appearance of your store
Powered by ajax
It has all the necessary elements: the basket, adding a product key in the basket of goods and orders the control panel.

== Installation ==

* Download plug-in .zip
* Download to Wordpress in the "Plugins"
* Activate
* Connect to the template (button, cart)
* Customize to fit your needs
